# TAU Hardware Design (KiCad)

> **TAU (Test Acquisition Unit)** receiver board designed in **KiCad** for high-speed optical communication applications.

The TAU receiver board implements the analog front-end of an optical communication receiver by converting the low-level current generated by a photodiode into a high-speed differential electrical signal suitable for FPGA-based Bit Error Rate (BER) testing.

## Signal Flow

The receiver is composed of four primary stages:

### Photodiode

The photodiode converts incoming optical energy into a proportional electrical current that serves as the input to the receiver.

### Transimpedance Amplifier (TIA)

The **LMH6629** high-speed operational amplifier converts the photodiode current into a voltage while providing the initial stage of amplification.

### Limiting Amplifier / LVDS Interface

The **MAX3747** amplifies the analog voltage and generates a differential LVDS output for high-speed digital communication.

### Output Stage

The conditioned differential signal is routed through coaxial connectors for measurement, FPGA interfacing, and BER testing.

## Hardware Features

- High-speed analog signal path
- LMH6629 Transimpedance Amplifier
- MAX3747 Limiting Amplifier
- Dual LM317 adjustable voltage regulators
- Photodiode input interface
- Differential coaxial outputs
- Multiple debugging and measurement test points
- Two-layer PCB layout
- Manufacturing-ready design
