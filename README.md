# Engineering Portfolio

> A collection of electronic hardware design projects developed using both **KiCad** and **Cadence OrCAD Capture/PSpice**.

This repository is organized into two independent design environments:

- **KiCad** — Open-source PCB design projects
- **PSpice** — Cadence OrCAD Capture schematics and SPICE simulations

Each directory contains its own documentation describing the projects, file structure, and how to open the designs.
